# file rename

ファイル名をまとめて整えるための小さな Python スクリプト集です。

## 含まれているスクリプト

### `先頭の空白削除.py`

- 指定フォルダ以下を再帰的に走査します
- ファイル名の先頭にある空白を削除します
- フォルダ名は変更せず、ファイル名だけを対象にしています

### `英単語の間に空白.py`

- 指定フォルダ以下を再帰的に走査します
- `wordninja` を使って英単語の区切りを推定し、ファイル名に空白を入れます
- 例: `bestqualitygirl.png` -> `best quality girl.png`

## 必要環境

- Python 3.x
- `wordninja` (`英単語の間に空白.py` で使用)

インストール例:

```bash
pip install wordninja
```

## 使い方

各スクリプト内の `folder_path` を、自分が処理したいフォルダに変更してから実行します。

実行例:

```bash
python 先頭の空白削除.py
python 英単語の間に空白.py
```

## 注意

- どちらもファイル名を直接変更します
- 実行前に対象フォルダのバックアップを取ることをおすすめします
- 同名ファイルが存在する場合や、意図しない単語分割が起きる場合があります
- 現在はパスをコード内に直接書く形式です。公開用に使うなら引数対応へ改善してもよいです
