# APNG作成ツール

PyQt5 ベースの GUI で、複数の画像から APNG を作成できるツールです。  
PNG / JPG / JPEG を並べて、フレーム順、表示時間、ループ設定を調整しながら APNG を出力できます。

## 主な機能

- 複数画像から APNG を作成
- ドラッグ＆ドロップで画像追加
- サムネイル付きでフレーム順を並べ替え
- フレーム表示時間の指定
- 無限ループ / 指定回数ループの切り替え
- 先頭画像サイズに合わせた一括リサイズ
- 作成した APNG のプレビュー
- 既存 APNG を読み込んでフレームを再編集
- 前回設定の保存

## 動作環境

- Python 3.10 以上推奨
- Windows 想定

## 必要パッケージ

```bash
pip install PyQt5 Pillow apng
```

## 起動方法

`apng_maker_gui.py` があるフォルダで次を実行します。

```bash
python apng_maker_gui.py
```

## 使い方

1. `画像追加` で画像を選ぶか、画像ファイルを一覧へドラッグ＆ドロップします。
2. サムネイルをドラッグしてフレーム順を並べ替えます。
3. `フレーム秒数` で 1 コマあたりの表示時間を設定します。
4. 必要に応じてループ設定を変更します。
5. `最初の画像サイズにリサイズ` をオンにすると、全画像を先頭画像のサイズに揃えて出力します。
6. 出力先フォルダを指定します。未指定の場合は先頭画像のフォルダに保存されます。
7. `APNG作成` を押すと APNG を生成します。

出力ファイル名は、先頭画像のファイル名をベースにした `先頭画像名.png` になります。

## 既存 APNG の編集

`プレビュー読込` から APNG を開くと、フレームを抽出して一覧に読み込みます。  
その後は通常の画像列と同じように並べ替えや設定変更を行い、再度 APNG として保存できます。

## 対応形式

入力:

- `.png`
- `.jpg`
- `.jpeg`

出力:

- `.png` 形式の APNG

## 設定ファイル

同じフォルダにある `apng_maker_settings.json` に以下の設定が保存されます。

- フレーム表示時間
- ループ設定
- リサイズ設定
- 出力先フォルダ

## 補足

- 画像が 2 枚未満の場合は APNG を作成できません。
- 入力画像サイズが異なる場合でも、リサイズをオンにすれば統一して出力できます。
- APNG 読み込み時は一時フォルダにフレームを展開し、終了時に削除されます。
