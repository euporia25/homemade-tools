# YouTube Auto Skip

YouTube の広告スキップボタンを自動で押すための Chrome 拡張です。  
通常の DOM クリックに加えて、必要に応じて `chrome.debugger` を使った実クリックも行い、スキップ操作が通りにくいケースにも対応しています。

## 特徴

- YouTube の広告スキップボタンを自動検出
- ボタンが表示されたら自動でクリック
- MutationObserver と定期チェックを併用して検出漏れを減らす
- 日本語・英語のスキップラベルに対応
- スキップ後に広告が残っている場合は再試行

## 動作ファイル

- `manifest.json`
  Chrome 拡張の設定ファイルです。
- `content.js`
  YouTube ページ上でスキップボタンを検出し、クリック処理を実行します。
- `background.js`
  `chrome.debugger` を使って、対象タブに対して実際のマウスクリックイベントを送ります。

## インストール方法

1. このフォルダを手元に置きます。
2. Chrome で `chrome://extensions/` を開きます。
3. 右上の「デベロッパー モード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックします。
5. このプロジェクトのフォルダを選択します。

## 使い方

拡張を読み込んだ状態で YouTube を開くと、自動で監視が始まります。  
広告のスキップボタンが表示されると、自動でクリックされます。

## 権限について

この拡張は `debugger` 権限を使用します。

これは YouTube 側の UI 変化やクリック制御によって通常の `button.click()` だけでは反応しない場合に、Chrome Debugger API 経由で実際のマウス操作に近いイベントを送るためです。

そのため、初回利用時や拡張の挙動によっては、Chrome 側で権限に関する注意表示が出ることがあります。

## 注意点

- 対象は `https://www.youtube.com/*` のみです
- YouTube の UI 変更により、将来的に動かなくなる可能性があります
- 広告そのものをブロックするものではなく、「スキップ可能になった広告」を自動でスキップする拡張です
